## 快速上手

### 引入

```
import {Button, Tabs, Switch, Dialog} from "coriander-ui";

import 'coriander-ui/dist/lib/coriander.esm.css'
```

  
### Vue 单文件组件代码示例

```
<template>
  <div>
    <Button>按钮</Button>
  </div>
</template>

<script>
import {Button, Tabs, Switch, Dialog} from "coriander-ui"
import 'coriander-ui/dist/lib/coriander.esm.css'

export default {
  components: {Button}
}
</script>
```
